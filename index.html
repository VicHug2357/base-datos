Historia

Orígenes

Las bases de datos existen desde la Antigüedad. Hablando de bases de datos ligadas a la informática, se remonta a 1884 con Herman Hollerith, con el tabulador electromagnético de tarjetas perforadas para resumen de información y posteriormente en apoyo a la contabilidad.

  • La década de los 50 y los 60
    •
    1. Cintas magnéticas
    2. Discos duros
    3. inició la primera generación de bases de datos de red (CODASYL) y las jerárquicas (IMS)
  • Los años 70 y 80
    •
    1. Edgar Frank Codd, científico informático inglés, definió el modelo relacional (12 reglas)
    2. Lawrence “Larry” Ellison, desarrolla el Relational Software System (Oracle Corporation)
    3. Lenguaje de consulta estructurado (SQL – Structured Query Language)
  • A partir de los 90
    •
    1. Bases de datos orientas a objetos
    2. Aparecen herramientas como Excel y Access
    3. Nuevas definiciones de modelos de base de datos

¿Qué es un dato?

Un dato es la representación de una variable que puede ser cuantitativa o cualitativa que indica un valor que se le asigna a las cosas y se representa a través de una secuencia de símbolos, números o letras.

Dato en informática es la expresión general que describe aquellas características de la entidad sobre la que opera.

El dato es el elemento primario de la información elemento primario de la información conformado por símbolos (letras, números, dibujos, señas, gestos) que reunidos pueden cobrar significado.

Solo o aislado el dato no posee relevancia.

La información

La información está constituida por un grupo de datos supervisados y ordenados, que sirven para construir un mensaje basado en un cierto fenómeno o ente.

La información permite resolver problemas y tomar decisiones, ya que su aprovechamiento racional es la base del conocimiento.

Informar, consiste en la transmisión de los datos

La información es el conjunto de datos, procesados y ordenados para su comprensión, que aportan nuevos conocimientos. Puede ser a un individuo o sistema y sobre un asunto, materia, fenómeno o ente determinado.

Sistema de información (SI)

Se define como un sistema de información (SI) al sistema que toma, almacena y transmite información.

En términos sencillos, es la interacción de personas y ordenadores para procesar e interpretar los datos.

El término también define el software que se utiliza para las bases de datos.

El objetivo principal de cualquier sistema de información es apoyar las operaciones, la gestión, la toma de decisiones, el control y la coordinación de una organización

Estos son conjuntos de elementos interrelacionados que recopilan, procesan, almacenan y distribuyen información
Tipos de sistemas de información (SI)

  • Procesamiento automatizado de las transacciones
  • Apoyo a la decisión
  • Gestión del conocimiento
  • Sistema de gestión de bases de datos (SGBD)

¿Qué es una base de datos?

Una base de datos es una recopilación organizada de información o datos estructurados, que normalmente se almacena de forma electrónica en un sistema informático.

Se llama base de datos, también banco de datos, (Base de conocimientos) a un conjunto de información perteneciente a un mismo contexto, que se encuentra ordenado de modo sistemático para su posterior recuperación, análisis y/o transmisión.

Una Base de Datos es una herramienta que funciona como “almacén”, es decir, guarda grandes cantidades de información de forma organizada para poder encontrarla y utilizarla de manera fácil y ordenada

Las bases de datos tienen varias funciones empresariales importantes.

  1. Amacenamiento de datos
  2. Recuperación de datos
  3. Manipulación de datos
  4. Seguridad de datos
  5. Integridad de datos
  6. Escalabilidad
  7. Confiabilidad

Elementos principales en las base de datos

  • Entidad (Objeto)
    • Atributo (Característica)
  • Asociaciones entre entidades (Relación)
  • Entidades (Objeto)
    •

    Son objetos o conceptos del mundo real o abstarctos, que son representados en una base de datos.

    Por ejemplo
    En una base de datos de una empresa, las entidades podrían ser empleados, clientes, productos, entre otros.

  • Atributos (Caracteristicas)
    •

    Son las características o propiedades que describen a una entidad (Objeto) en una base de datos.

    Por ejemplo
    En una entidad "empleado" los atributos de la entidad podrían ser el nombre, la edad, el cargo, el salario, entre otros.

  • Asociaciones entre entidades (Relaciones)
    •

    Son relaciones que existen entre las entidades de una base de datos. Estas asociaciones pueden ser de diferentes tipos, como uno a uno, uno a muchos, muchos a muchos.
    Por ejemplo
    En una base de datos de una tienda, la asociación entre la entidad "cliente" y la entidad "compra" sería de uno a muchos, ya que un cliente puede realizar varias compras.

  • En resumen
    •

    Las bases de datos son fundamentales para el almacenamiento, recuperación, manipulación y protección de la información, facilitando la gestión eficiente de los datos en entornos empresariales y de aplicaciones informáticas.

  • Dato como campo
    •

  • Dato: es el elemento primario de la información conformado por símbolos (letras, números, dibujos, señas, gestos) que reunidos pueden cobrar significado.

    • Pero el dato solo o aislado no posee relevancia.

  • Campo: El llamado campo o atributo, es simplemente la abstracción del dato, representan las columnas que conforman nuestra tabla o entidad de base de datos, la característica única, es que todos los elementos que estén en el campo deben de ser de un tipo de dato especifico.

    Representan caracteristicas abstractas de un objeto

    • La colección de campos representan los registros

    Los registros son la recopilación de todos los campos en una fila, filas que conforman nuestra tabla o entidad de base de datos, representando la información de contexto empresarial

    Tambien llamdos archivo

    El archivo es todo lo que conforma la tabla, entidad u objeto de las bases de datos

  • El hardware de una base de datos
    •

    El hardware de una base de datos se refiere a los componentes físicos que se utilizan para almacenar y procesar los datos. Esto incluye servidores, discos duros, unidades de cinta y otros dispositivos de almacenamiento. El hardware de una base de datos debe ser lo suficientemente potente para manejar grandes cantidades de datos y permitir un acceso rápido a ellos.

  • Software de base de datos

    • Este se utiliza para crear, editar y mantener registros y archivos de bases de datos, "la entrada de datos en una APP", esto permite, la edición, la actualización, el almacenamiento de los datos, para la creación de informes, copias de seguridad, y el control de acceso múltiple.

    En ocasiones, también se denomina, "sistema de gestión de bases de datos" (DBMS o SGDB).

    El proceso es el siguiente: el desarrollador escribe un comando específico usando un lenguaje de acceso y lo envía a la base de datos por medio del SGBD.

    Este se encargará de procesar y generar la información a fin de ofrecer un resultado en un formato que el usuario pueda leer.
